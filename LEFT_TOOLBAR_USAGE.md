# 伯努利模擬器 - 左側工具列版本使用說明

## 🚀 如何運行

```bash
cd /mnt/c/Users/iven8/pygamelin
python3 bernoulli_dual_view_left_toolbar.py
```

## 🎯 主要特色

### 1. 視窗控制按鈕 (右上角)
- 🔴 **關閉按鈕** (紅色) - 關閉程式
- 🟡 **最小化按鈕** (黃色) - 最小化到工作列  
- 🔵 **縮放按鈕** (藍色) - 循環切換視窗大小

### 2. 左側工具列
- **控制面板**: 所有參數調整滑桿
- **物理數據**: 即時顯示物理參數
- **視窗資訊**: 顯示當前視窗大小

### 3. 右側雙視圖
- **XY 平面視圖** (左側) - 顯示水平和垂直運動
- **XZ 平面視圖** (右側) - 顯示水平和深度運動

## 🎮 操作方式

### 視窗控制
- **點擊縮放按鈕**: 循環切換 5 種視窗大小
  - 小視窗 (800×600)
  - 中視窗 (1024×768) 
  - 大視窗 (1280×720)
  - 預設大小 (1400×800)
  - 超大視窗 (1600×900)

### 球體控制
- **拖拽球體**: 在任一視圖中點擊並拖拽球體
- **顏色含義**:
  - 🟢 綠色 = 上升 (升力 > 重力)
  - 🟡 黃色 = 平衡 (升力 ≈ 重力)
  - 🔴 紅色 = 下降 (升力 < 重力)

### 參數調整 (左側滑桿)
- **風速**: -50 到 50 m/s (右正左負)
- **風向**: 0 到 360 度
- **垂直風**: -20 到 20 m/s
- **球體半徑**: 20 到 80 像素 (影響質量)
- **推力**: -1000 到 1000 N (額外升力)
- **側向係數**: 0 到 1.0 (側向力強度)

### 鍵盤快捷鍵
- `V` - 切換風速向量顯示/隱藏
- `R` - 切換視窗大小 (與縮放按鈕相同)
- `ESC` - 退出程式

## 📊 物理數據顯示

左側工具列下方顯示即時物理數據：
- **位置**: 球體的 3D 座標 (X, Y, Z)
- **速度**: 球體的 3D 速度向量
- **升力**: 當前升力大小 (牛頓)
- **側力**: 側向力大小 (牛頓)
- **壓差**: 上下壓力差 (千帕)

## 🌬️ 風速向量

當啟用風速向量顯示時：
- **紅色箭頭**: X 方向風速 (水平)
- **綠色箭頭**: Y 方向風速 (垂直)
- **藍色箭頭**: Z 方向風速 (深度)

- **響應式設計**: 所有元素隨視窗大小自動縮放
